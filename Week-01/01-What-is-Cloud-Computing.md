# What is Cloud Computing?

## Definition

Cloud computing is the on-demand delivery of IT resources over the internet using a pay-as-you-go pricing model. Instead of purchasing and maintaining physical infrastructure, organizations can access computing resources whenever needed.

---

## Why Cloud Computing?

Traditional IT infrastructure requires organizations to buy, maintain, and upgrade physical servers. Cloud computing eliminates these challenges by providing flexible and scalable resources over the internet.

---

## Benefits

- **Scalability** – Increase or decrease resources based on demand.
- **High Availability** – Services remain available with minimal downtime.
- **Cost Effective** – Pay only for the resources you use.
- **Global Reach** – Deploy applications closer to users around the world.

---

## AWS Global Infrastructure

AWS delivers cloud services through:

- **Regions** – Geographic locations containing AWS data centers.
- **Availability Zones (AZs)** – Multiple isolated data centers within a Region.
- **Edge Locations** – Used to deliver content with low latency.

---

## Shared Responsibility Model

AWS follows a Shared Responsibility Model:

- **AWS is responsible for the security *of* the cloud**, including physical infrastructure, networking, and hardware.
- **Customers are responsible for security *in* the cloud**, including data, identities, applications, and access permissions.

---

## Key Takeaways

- Cloud computing provides IT resources over the internet.
- Resources are available on demand.
- Customers pay only for what they use.
- AWS is one of the world's leading cloud service providers.
